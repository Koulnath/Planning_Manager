🌿 Règle d'Or : Git Workflow

Ne codez jamais sur la branche main. Suivez ce cycle pour chaque tâche :

Étape 1
git checkout skjuv-conflits  # Ou votre branche dédiée
git pull origin main          # Toujours synchroniser
Étape 2

Codez votre fonctionnalité en local.

Étape 3
git add .
git commit -m "feat: ajout de la logique de [votre tâche]"
git push origin [votre-branche]
